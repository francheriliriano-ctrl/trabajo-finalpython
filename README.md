#  Trabajo Final Python


## Objetivo
Desarrollar un programa funcional aplicando buenas prácticas,
PEP8, PEP257 y principios del Zen de Python.

## Funcionalidades
- Agregar tareas
- Listar tareas
- Completar tareas

## Buenas prácticas
- Código modular
- Funciones simples y claras
- Docstrings según PEP 257
- Código legible siguiendo PEP8

## Zen de Python aplicado
"Simple is better than complex"
El proyecto prioriza simplicidad y claridad.

## Librerías externas
- Typer
- Rich

## Entorno virtual
Uso de venv para aislar dependencias y facilitar el trabajo colaborativo.

## Instalación
python -m venv venv
pip install -r requirements.txt

## Ejecución
python main.py agregar "Ejemplo"
python main.py listar
python main.py completar 0
